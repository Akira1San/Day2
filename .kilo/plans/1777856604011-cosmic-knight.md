# Plan: Add Added Videos and Blacklist Management to Custom Tag

## Goal
Enhance the `TagDialog` (Custom Tag editor) to feature-match `RandomFillDialog` by adding:
1. `added_videos` list - selective video picking from collection
2. Full blacklist management UI - separate list widget with load/save/remove buttons
3. Three-panel layout: Collection Videos | Added Videos | Blacklist

## Current State Analysis

### RandomFillDialog (Reference Implementation)
- Has `self.added_videos` - videos explicitly selected for playback
- Has `self.blacklist` list with management UI (list widget + buttons)
- Three-panel layout in `_create_video_list_section()` and `_create_blacklist_section()`

- `load_collection()` auto-loads blacklist and auto-adds non-blacklisted videos
- `get_tag()` returns `collection_videos=self.added_videos.copy()`

### TagDialog (Custom Tag - Current)
- Has `self.collection_videos` - all videos from collection
- Has `self.blacklist` but only as a simple list; no list widget or management buttons
- Single-panel layout: only "Videos in Collection" list
- No `added_videos` attribute - uses entire `collection_videos` directly
- No blacklist UI beyond a profile dropdown
- `get_tag()` returns `collection_videos=self.collection_videos.copy()` (all of them)

#### 5. Replace `load_collection()` (lines 185-197)
Rewrite to match RandomFillDialog's `load_collection()` logic:
```python
def load_collection(self, file_path: str, load_blacklist: bool = True):
    collection_videos, collection_info = load_collection_json(file_path)
    self.collection_path.setText(file_path)
    self.videos_list.clear()
    self.collection_videos = []
    self.added_videos = []
    
    if load_blacklist:
        self.blacklist = []
    
    collection_dir = Path(file_path).parent
    collection_stem = Path(file_path).stem
    
    # Auto-detect and load matching blacklist if load_blacklist=True
    if load_blacklist:
        blacklist_data = []
        blacklist_patterns = [f"{collection_stem}_blacklist.*", f"{collection_stem.replace('collections_', '')}_blacklist.*"]
        for search_dir in [collection_dir, Path.cwd()]:
            for pattern in blacklist_patterns:
                for bl_file in search_dir.glob(pattern):
                    blacklist_data = load_blacklist_json(str(bl_file))
                    break
                if blacklist_data:
                    break
            if blacklist_data:
                break
        self.blacklist = blacklist_data
    
    # Populate collection list
    for video in collection_videos:
        path = video.get('path', '')
        duration = video.get('duration', 0)
        video_data = {'path': path, 'duration': duration, 'name': get_video_display_name(video)}
        self.collection_videos.append(video_data)
        self.videos_list.addItem(f"{video_data['name']} ({format_duration(duration)})")
    
    # Populate blacklist list and auto-filter added_videos
    self.refresh_blacklist_list()
    
    # Auto-add non-blacklisted videos if config enabled
    if load_blacklist and get_randomfill_config():
        for video in self.collection_videos:
            path = video.get('path', '')
            if not any(b.get('path') == path for b in self.blacklist):
                self.added_videos.append(video.copy())
    else:
        # If not auto-add, start with empty added_videos
        pass
    
    self.refresh_added_list()
```

## Edge Cases to Handle
- Editing existing custom tag: restore `added_videos` and `blacklist`
- Loading collection with blacklist: auto-filter added_videos
- Changing blacklist profile: reload blacklist and re-filter added_videos
- Remove from added: if video in blacklist, keep it in blacklist (only remove from added)
- Auto-add config off: start with empty added_videos (user explicitly adds)
- videos_list selection shows info in video_info label
